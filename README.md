# MealSync AI
# Family Food Planner

## Project Summary

The Family Food Planner is a web application that helps families or groups create personalized meal plans based on their dietary needs and preferences. Users can enter information for multiple people, including allergies, dietary restrictions (such as vegetarian or gluten-free), food preferences, and other relevant details. The application combines this information to generate meal suggestions that accommodate everyone in the group.

The website is built with **HTML, CSS, and JavaScript** for the user interface and **Python (Flask)** for the backend. The backend securely communicates with an AI API to generate customized meal plans without exposing the API key to users.

## Technologies Used

* HTML
* CSS
* JavaScript
* Python (Flask)
* AI API

## Features

* Add multiple family or group members
* Record dietary restrictions and allergies
* Combine everyone's dietary needs into a single request
* Generate AI-powered meal plan recommendations
* Clean, user-friendly web interface

## Project Structure

* **Frontend:** HTML, CSS, and JavaScript files that handle the user interface and user interactions.
* **Backend:** Flask server that processes requests and communicates with the AI API.
* **GitHub:** Used for collaboration, version control, and sharing the project among team members.
